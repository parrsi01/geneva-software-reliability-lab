# Failure Taxonomy

## Short Definition
Failure taxonomy groups incident types to speed triage and prevention.

## Enterprise Context
Categorization improves escalation accuracy and recurring incident analysis.

## Categories Used Here
- Runtime crash
- Memory leak
- Race condition / deadlock
- API failure from bad config + weak validation
- CI pipeline failure
- Container crash loop

## Geneva Role Alignment
Supports multi-team handoffs and faster routing in complex organizations.
