# Lesson Research & Analysis Companion

Author: Simon Parris  
Date: 2026-02-26

Use this companion to extract interview-ready reasoning from each lab and incident.

## Analysis Framework

For each failure type, document:

1. Symptom pattern
2. Root cause classes
3. Evidence to collect first
4. Fastest safe rollback
5. Long-term prevention strategy

## Interview Translation

- "How would you debug a crash loop?"
- "How do you isolate a race condition?"
- "What evidence would you collect before patching a production incident?"

Answer each using a lab from this repo and specific commands/artifacts.

## Portfolio Extraction Ideas

- RCA write-ups
- incident timelines
- reproduction scripts
- postmortem summaries
- regression-test additions
